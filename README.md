# 집ZOOM

## "초등학생들이 원격 수업에서도 즐겁게 참여할 수 있도록 돕는 서비스"

### 👩‍💻 개발 동기
코로나 확진자 증가로 인해 원격 수업 이용률이 대폭 증가하였습니다.  

하지만, 원격 수업은 여러가지 불편한 점이 있습니다.  
1) 대면 수업 때보다 학생들의 수업 참여율이 떨어짐.
2) 선생님들이 학생들을 수업에 참여시키는 것 또한 힘듦.

저희는 이런 문제점들이 결국 초등학생들의 사회성 발달 저하까지 이어진다는 것이 우려되었습니다.  

그래서 **집ZOOM**을 개발하였습니다!

### 💁‍♀️ 주요 기능
집ZOOM은 클라이언트 프로그램과 웹사이트로 구성되어있습니다.  
또한, 클라이언트 프로그램은 ZOOM과 같은 원격 회의 시스템과 같이 사용할 수 있습니다.  
클라이언트 프로그램 github 주소 : https://github.com/kangminjugit/HouseZOOMClient
집ZOOM 사용자 매뉴얼 : [집ZOOM 사용 매뉴얼](HouseZOOM/집ZOOM 사용 매뉴얼.pdf)

#### 선생님/학생용 클라이언트 프로그램 
1. 졸음 방지 기능  
수업 시간 중 학생이 졸 때 선생님이 일일이 깨울 필요가 없습니다.  
학생 스스로 일어날 수 있도록 재밌는 잠 깨기 게임을 제공합니다.  
<img width="1051" alt="스크린샷 2022-06-26 오후 2 48 57" src="https://user-images.githubusercontent.com/62755858/175801423-fc8f07ee-092f-4428-9120-0b35b2f2c625.png">

2. 퀴즈 기능  
학생들은 제스쳐로 퀴즈를 풀면서 수업에 더 흥미를 느낄 수 있습니다.  
퀴즈를 맞추면 뱃지와 칭찬 포인트를 받을 수 있습니다.  
받은 칭찬 포인트는 원격 수업 화면에 나타나는 캐릭터를 꾸밀 때 사용할 수 있습니다.

3. 뱃지 기능    
수업 중 참여를 잘 한 학생에게 뱃지를 부여하여 학생들의 참여도를 높일 수 있습니다.  
뱃지는 원격 수업 화면에 나타납니다.

1. 캐릭터 상점  
수업 중 받은 칭찬 포인트로 캐릭터를 꾸미는 아이템을 구입할 수 있습니다.

구입한 아이템들로 자신의 캐릭터를 개성 있게 꾸밀 수 있습니다.
